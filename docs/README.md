# GP2 Documentation

This folder contains the canonical documentation for the GP2 smart-helmet prototype.

## Contents

- Component reference: `COMPONENTS.md`
- Testing guide + expected outputs: `TESTING.md`

## Quick start (tests)

```bash
cd unit-tests
python -m unittest -v
```

If `numpy` is missing:

```bash
python -m pip install numpy
```
